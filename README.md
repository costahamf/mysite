# CRM для рекрутеров курьеров (PHP + MySQL)

Простая CRM-система для рекрутеров, рассчитанная на обычный shared-хостинг (PHP 8+, MySQL).

## Функционал

- Регистрация рекрутера (`/register`) с проверкой уникальности телефона/email.
- Авторизация (`/login`) через `password_verify` и PHP Session.
- Роли: `recruiter`, `admin`.
- Кабинет рекрутера (`/dashboard`):
  - добавление/редактирование/удаление курьеров,
  - таблица только своих курьеров,
  - счетчик курьеров и общий заработок.
- Админ-панель (`/admin`):
  - список рекрутеров,
  - список всех курьеров,
  - редактирование заказов/вознаграждения/статуса.
- Безопасность:
  - prepared statements,
  - проверка сессии,
  - защита админ-маршрутов,
  - logout.

## Установка на хостинг (например, Sweb.ru)

1. Создайте базу MySQL в панели хостинга.
2. Импортируйте файл `install.sql` через phpMyAdmin.
3. Загрузите файлы проекта в корень сайта (`public_html` или аналог).
4. Откройте `config/db.php` и укажите данные БД:
   - `DB_HOST`
   - `DB_NAME`
   - `DB_USER`
   - `DB_PASS`
5. Убедитесь, что на хостинге включен `mod_rewrite` (для `.htaccess`).
6. Перейдите на сайт:
   - регистрация рекрутера: `/register`
   - вход: `/login`

## Доступ админа (по умолчанию)

- Email: `admin@example.com`
- Пароль: `admin12345`

> После первого входа рекомендуется заменить данные админа в базе.

## Структура проекта

- `config/` — подключение БД и инициализация.
- `auth/` — регистрация/вход/выход.
- `dashboard/` — кабинет рекрутера и управление курьерами.
- `admin/` — админ-панель.
- `assets/` — CSS/JS.
- `install.sql` — схема и стартовые данные.
